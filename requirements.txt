# ========================================
# Installation Instructions
# ========================================
#
# Minimal Installation (Core features only):
#   pip install requests
#
# Standard Installation (with document processing):
#   pip install -r requirements.txt
#
# Windows (for full features including Registry/Event Logs):
#   pip install -r requirements.txt
#   python Scripts\pywin32_postinstall.py -install
#
# macOS/Linux (without Windows-only features):
#   pip3 install requests PyPDF2 python-docx openpyxl Pillow pytesseract opencv-python pandas
#
# For Document Scanning (PII Detection in PDFs/Word/Excel/Images/Videos):
#   pip install PyPDF2 python-docx openpyxl Pillow pytesseract opencv-python pandas
#   # Also install Tesseract-OCR binary:
#   # Windows: Download from https://github.com/UB-Mannheim/tesseract/wiki
#   # macOS: brew install tesseract
#   # Linux: sudo apt-get install tesseract-ocr
#
# Upgrade existing packages:
#   pip install --upgrade -r requirements.txt
#
# Install in virtual environment:
#   python -m venv venv
#   source venv/bin/activate  # On Windows: venv\Scripts\activate
#   pip install -r requirements.txt
#
# Verify installation:
#   python check_dependencies.py

# ========================================
# Troubleshooting
# ========================================
#
# Issue: "pywin32 module not found" on Windows
#   Solution:
#     pip uninstall pywin32
#     pip install pywin32
#     python Scripts\pywin32_postinstall.py -install
#
# Issue: Permission errors during installation
#   Windows: Run Command Prompt as Administrator
#   macOS/Linux: Use sudo pip3 install
#
# Issue: SSL certificate errors
#   Solution:
#     pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org -r requirements.txt
#
# Issue: Outdated pip
#   Solution:
#     python -m pip install --upgrade pip
#     pip install -r requirements.txt
